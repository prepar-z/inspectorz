# inspectorz
Java 代码审计 idea 插件
## 介绍
- inspectorz 为 idea 静态代码扫描插件，可一键搜索出所有 sink 点
- 在原有基础上，补充添加了多个规则，例如权限绕过，rpc 调用等风险点
- 原作者 KimJun
- 原版插件链接
- https://github.com/KimJun1010/inspector
